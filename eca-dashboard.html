Cloud Scheduling Simulation Results

Energy–Carbon Aware (ECA) Scheduler

This dashboard compares four cloud task scheduling strategies — Round Robin, a Baseline Q-Learning agent, an Energy-Aware heuristic, and the proposed ECA Scheduler — across light, medium, and heavy workloads. ECA selects virtual machines using real-time carbon intensity signals rather than energy consumption alone, aiming to cut emissions without sacrificing execution speed. Select a workload and scheduler below to view the run summary, then scroll down for full comparison, workload-scaling, and per-VM carbon analyses.
